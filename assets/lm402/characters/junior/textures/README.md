# Textures

Store texture sources and baked maps here.

Suggested subfolders:

- `source/` for paint files and layered sources
- `bakes/` for baked normal/AO/mask maps
- `runtime/` for flattened runtime maps

Keep the slot names aligned with the main junior README.

Recommended runtime slots:

- `skin_face_basecolor`
- `skin_face_normal`
- `skin_face_roughness`
- `hair_basecolor`
- `hair_alpha`
- `hair_normal`
- `shirt_white_basecolor`
- `shirt_white_normal`
- `shirt_white_roughness`
- `shorts_denim_basecolor`
- `shorts_denim_normal`
- `shorts_denim_roughness`
